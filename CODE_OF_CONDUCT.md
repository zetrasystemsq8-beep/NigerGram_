# Contributor Covenant Code of Conduct

This project adopts the Contributor Covenant to provide a welcoming and respectful community. All participants are expected to abide by this code.

Key points
- Be respectful and considerate in language and actions.
- No harassment, discrimination, or abusive behavior.
- Respect others' time and effort; be constructive in feedback.

Reporting
- If you experience or witness unacceptable behavior, please open an issue or contact the repository maintainers.

For the full text of the Contributor Covenant, see: https://www.contributor-covenant.org/version/2/0/code_of_conduct/
